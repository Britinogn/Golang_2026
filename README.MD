# Udemy Go Practical Projects

This repository contains a collection of practical Go programming projects and examples from Udemy courses. Each folder represents a different project or concept demonstration.

## Projects

### go-bank
A simple banking application demonstrating file operations, user input, and basic banking functionalities.
- `bank.go`: Main banking logic
- `communication.go`: User communication functions
- `fileops/`: File operation utilities
- `output/`: Contains balance data

### go-practical-project
A profit calculator application.
- `app.go`: Main application entry point
- `profit_calculator.go`: Profit calculation logic
- `output/`: Contains calculation results

### go-project
An investment calculator project.
- `investment_calculator.go`: Investment calculation implementation

### Pointers
Examples and demonstrations of Go pointer usage.
- `pointer.go`: Pointer examples and exercises

### structs
Examples and demonstrations of Go struct usage.
- `structs.go`: Struct examples and exercises

## Getting Started

Each project can be run independently. Navigate to the respective folder and use `go run` to execute the main files.

For example:
```bash
cd go-bank
go run bank.go
```

## Prerequisites

- Go 1.16 or higher installed on your system
